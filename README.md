# Zepto-PowerBI-Dashboard

# 📊 Zepto Sales & Profit Analysis Dashboard

## 🚀 Project Overview

This project focuses on analyzing retail sales data to generate meaningful business insights. The dashboard is built using Power BI and highlights key performance indicators such as revenue, profit, orders, and sales performance.

---

## 🎯 Objective

* Analyze product-wise and category-wise sales performance
* Identify top-performing products and cities
* Understand profit trends and margins

---

## 📁 Dataset Description

The dataset contains the following columns:

* product_name
* category
* city
* original_price
* selling_price
* order_count

### 🔧 Derived Columns

* total_revenue = selling_price × order_count
* total_profit = total_revenue - (original_price × order_count)
* profit_per_order
* profit_margin %

---

## 📊 Dashboard Features

### 🔹 KPIs

* Total Revenue
* Total Profit
* Total Orders
* Profit Margin %
* Average Selling Price

### 🔹 Visualizations

* Product-wise Revenue & Profit
* Category-wise Revenue Distribution
* City-wise Sales Analysis
* Order Trend by Products

### 🔹 Filters (Slicers)

* City
* Category

---

## 🧠 Key Insights

* A few top products contribute to the majority of revenue
* Certain cities outperform others in terms of sales
* Snacks category generates significant revenue
* Profit margins remain stable across categories

---

## 🛠 Tools Used

* Power BI
* Python (Pandas for data cleaning)
* Excel/CSV dataset

---

## 📌 Conclusion

This dashboard helps in understanding business performance and supports data-driven decision-making. It can be used to identify growth opportunities and optimize sales strategies.
